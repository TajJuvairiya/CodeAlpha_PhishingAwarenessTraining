# CodeAlpha_PhishingAwarenessTraining

Phishing Awareness Training 🛡️
A cybersecurity awareness presentation created as part of the CodeAlpha Cybersecurity Internship (Task 2).

📋 About
This training module educates users on how to recognize and avoid phishing attacks. It covers real-world examples, common tactics used by attackers, and best practices to stay safe online.

Slide     Topic

6         Real-World Examples (Google, Sony Pictures, Twitter)
7         Best Practices — DO vs DON'T
8–10      Interactive Quiz (3 Questions with Answers)
11        Closing Slide

🧠 Topics Covered
How to identify phishing emails
